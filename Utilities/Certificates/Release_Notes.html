Purpose

This utility provides a set of functions to generate a X509 certificate (DUA User Free Usage and/or DUA User Licensed Usage, where DUA stands for Device Unique Authentication). It is used on devices with key provisioned in system flash.

The interfaces are :

  • UTIL_CERT_GetCertificateSize : It retrieves the length of the certificate X509
  • UTIL_CERT_GetCertificate : It builds an X509 certificate from a template and information stored in system flash

This utility contains the ROOT CA "DUA User FU" Certificate in pem format. This certificate can be used to verify the certificate "DUA User Free Usage" generated by the utility.
